# ホームページを開く方法

## パソコンで開く場合

**手順1:** 画面下のタスクバー（またはデスクトップ）にある「Microsoft Edge」や「Google Chrome」などのアイコンをクリックします。

> 「インターネットを見るためのアプリ」のことを**ブラウザ**といいます。

**手順2:** 画面上部にある細長い入力欄（アドレスバー）に、次のアドレスを入力してEnterキーを押します。

```
https://www.miyosino.com/
```

**手順3:** ホームページが表示されます。

---

## スマートフォンで開く場合

**手順1:** スマートフォンの画面にある「Safari」（iPhoneの場合）または「Chrome」（Androidの場合）のアイコンをタップします。

**手順2:** 画面上部または下部のアドレス入力欄をタップし、次のアドレスを入力して「開く」または「移動」をタップします。

```
https://www.miyosino.com/
```

**手順3:** ホームページが表示されます。

---

## ブックマーク（お気に入り）に登録する方法

毎回アドレスを入力するのは大変なので、ブックマークに登録しておくと便利です。

### パソコン（Google Chrome）の場合
ホームページを開いた状態で、アドレスバーの右端にある「☆（星）」マークをクリックします。

### スマートフォン（iPhone・Safari）の場合
ホームページを開いた状態で、画面下の「□から↑が出ているマーク」をタップし、「ブックマークに追加」をタップします。
