# ホームページとは

## 「ホームページ」とは

**ホームページ**とは、インターネット上に公開されている「ウェブサイト」のことです。

パソコンやスマートフォン（スマホ）から、いつでも無料でご覧いただけます。

---

## かわつる三芳野団地のホームページ

かわつる三芳野団地には、公式のホームページがあります。

> **https://www.k-miyosino.com/**  
> ※ブックマーク（お気に入り）に登録しておくと便利です

---

## このホームページの目的

このホームページは、2つの目的のために運営されています。

**① 団地外の方への情報発信**  
団地への入居を検討している方に向けて、団地の特徴・コミュニティ・施設・アクセスなどの情報を発信しています。

**② 組合員の生活をより快適にする**  
すでに団地にお住まいの組合員に向けて、お知らせ・回覧板・イベント情報などをオンラインで提供しています。

---

## ホームページでできること

### 団地外の方向け（ログイン不要）

| できること | 説明 |
|-----------|------|
| 団地の情報を見る | 団地の特徴・コミュニティ・共有施設などの情報を確認できます |
| アクセス方法を調べる | 団地への行き方をご確認いただけます |
| お問い合わせをする | 管理組合への連絡先（電話・窓口）を確認できます |

### 組合員向け（要ログイン）

| できること | 説明 |
|-----------|------|
| お知らせを見る | 組合員専用のお知らせが確認できます |
| 回覧板を見る | 回覧板・配布資料をダウンロードできます |
| イベント情報を見る | 団地のイベント情報を確認できます |
| 会議情報を見る | 総会や街区班長会議などの会議資料や議事録を確認できます |
| 申請書をダウンロードする | 各種申請書をダウンロードできます |
| グリーンウェルネスを見る | 管理規約をダウンロード・確認できます |

ログインが必要な機能については、[ログインの方法](../04-login/how-to-login.md)をご参照ください。
