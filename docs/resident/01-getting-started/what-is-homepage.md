# ホームページとは

**ホームページ**とは、インターネット上に公開されている「団地の公式ウェブサイト」のことです。

パソコンやスマートフォン（スマホ）から、いつでも無料でご覧いただけます。

---

## ホームページでできること

| できること | 説明 |
|-----------|------|
| 団地の情報を見る | 団地の特徴・コミュニティ・共有施設などの情報を確認できます |
| アクセス方法を調べる | 団地への行き方をご確認いただけます |
| お問い合わせをする | 管理組合への連絡フォームが使えます |
| お知らせを見る（要ログイン） | 組合員専用のお知らせが確認できます |
| 回覧板を見る（要ログイン） | 回覧板・配布資料をダウンロードできます |
| イベント情報を見る（要ログイン） | 団地のイベント情報を確認できます |

「要ログイン」と書かれている機能は、**組合員専用ページ**のものです。ログインが必要です（→ [ログインの方法](../04-login/how-to-login.md)）。
