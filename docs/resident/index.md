# 組合員向けマニュアル

かわつる三芳野団地ホームページの使いかたを説明しています。

## このマニュアルでわかること

- ホームページをパソコンやスマートフォンで開く方法
- 画面の見かた（メニューの使いかたなど）
- 各ページに何が書いてあるか
- 組合員専用ページへのログイン方法
- 組合員専用ページの使いかた（お知らせ・回覧板・イベントなど）

## はじめてご覧になる方へ

まず「[ホームページとは](01-getting-started/what-is-homepage.md)」から読み始めることをおすすめします。

ログイン方法だけ知りたい方は「[ログインの方法](04-login/how-to-login.md)」をご覧ください。
