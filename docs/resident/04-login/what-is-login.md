# 組合員専用ページとは

「**組合員専用ページ**」は、かわつる三芳野団地の組合員だけがご覧になれる特別なページです。

---

## 組合員専用ページでできること

| ページ名 | 内容 |
|---------|------|
| お知らせ | 管理組合からの大切なお知らせ |
| 回覧板・配布資料 | デジタル回覧板や配布資料のダウンロード |
| イベント情報 | 団地内のイベント・行事の案内 |
| 会議情報 | 総会・理事会などの会議情報 |
| 各種申請 | 各種申請書のダウンロード |
| グリーンウェルネス | 緑化・健康活動に関する情報 |

---

## ログインとは

**ログイン**とは、「自分が組合員であることを確認する操作」のことです。

ログインには**ID（ユーザー名）**と**パスワード**が必要です。

> IDとパスワードは、管理組合事務局から配布された書類に記載されています。  
> 書類が見当たらない場合は [問い合わせ先](../06-trouble/contact-support.md) にご連絡ください。
