# アプリ一覧と用途

Kintoneにログイン後、トップページに使用するアプリの一覧が表示されます。

## ホームページで使用するアプリ

| アプリ名 | ホームページの表示場所 | 主な操作 |
|---------|------------------|---------|
| お知らせ | 組合員専用 → お知らせ | 追加・編集・削除 |
| 回覧板 | 組合員専用 → 回覧板・配布資料 | 追加・編集・削除 |
| イベント | 組合員専用 → イベント情報 | 追加・編集・削除 |
| 会議情報 | 組合員専用 → 会議情報 | 追加・編集・削除 |
| 申請書 | 組合員専用 → 各種申請 | 追加・編集・削除 |
| グリーンウェルネス | 組合員専用 → グリーンウェルネス | 追加・編集・削除 |

## アプリの開き方

1. Kintoneトップページの「アプリ」一覧から操作したいアプリをクリックします
2. アプリのレコード一覧が表示されます

## 重要: 「公開」設定について

多くのアプリには**「公開/非公開」を切り替えるフィールド**があります。  
このフィールドを「**公開**」に設定しないと、ホームページに表示されません。  
詳しくは [よくある間違いと確認方法](../08-tips/common-mistakes.md) を参照してください。
