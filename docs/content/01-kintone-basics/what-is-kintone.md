# Kintoneとは

**Kintone（キントーン）** はサイボウズ株式会社が提供するクラウド型のデータ管理サービスです。

## Kintoneの役割

かわつる三芳野団地のホームページでは、Kintoneをコンテンツの保存場所として使っています。

- Kintoneにデータを追加/編集/削除すると → **ホームページに自動的に反映されます**
- Kintoneの操作にプログラミングの知識は不要です

## 操作のイメージ

Kintoneの操作はExcelの表に似ています。

- **アプリ** = Excelファイル（お知らせ用・回覧板用など用途ごとに分かれています）
- **レコード** = Excelの1行（1件のお知らせや回覧板のデータ）
- **フィールド** = Excelの1列（タイトル・日付・内容などの項目）
