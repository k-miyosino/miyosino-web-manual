# Kintoneへのログイン

## 準備するもの

- **ログインID**: 事務局またはIT委員会から通知されたID
- **パスワード**: 同様に通知されたパスワード

## ログイン手順

**手順1:** ブラウザで以下のURLを開きます。

```
https://k-miyosino.cybozu.com/
```

**手順2:** ログインIDを入力します。

**手順3:** パスワードを入力します。

**手順4:** 「ログイン」ボタンをクリックします。

**手順5:** Kintoneのトップページ（ポータル）が表示されればログイン完了です。

---

## ログアウト方法

作業が終わったら必ずログアウトしてください。

1. 画面右上のユーザーアイコン（またはユーザー名）をクリック
2. 「ログアウト」をクリック

---

## ログインできない場合

- IDやパスワードを忘れた場合はIT委員会にご連絡ください
