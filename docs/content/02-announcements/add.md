# お知らせを追加する

## 手順

**手順1:** Kintoneの「**お知らせ**」アプリを開きます。

**手順2:** 右上の「**+ 新しいレコードを追加**」ボタンをクリックします。

**手順3:** 各フィールドを入力します。

| フィールド名 | 入力内容 | 必須 |
|------------|---------|------|
| タイトル | お知らせのタイトル | ○ |
| 内容 | お知らせの本文 | ○ |
| 日付 | 公開日（例: 2026-04-01） | ○ |
| 重要度 | 1=重要（赤）/ 2=推奨（オレンジ）/ 3=通常（青） | ○ |
| 公開設定 | 「**公開**」を選択（必須） | ○ |
| 添付ファイル | PDFなどを添付する場合 | - |

> **重要:** 「公開設定」を「公開」にしないとホームページに表示されません。

**手順4:** 入力が終わったら「**保存**」ボタンをクリックします。

**手順5:** ホームページの「お知らせ」ページを開き、追加した内容が表示されているか確認します。
