# お知らせを削除する

> **注意:** 削除したレコードは元に戻せません。間違いを防ぐため、削除の代わりに「非公開」に設定することを推奨します。

## 非公開にする方法（推奨）

**手順1:** 対象のお知らせレコードを編集します。

**手順2:** 「公開設定」フィールドを「**非公開**」に変更して保存します。

これでホームページから非表示になりますが、Kintone上にはデータが残ります。

---

## 完全に削除する方法

**手順1:** Kintoneの「**お知らせ**」アプリを開きます。

**手順2:** 削除したいお知らせの行の左端にあるチェックボックスをクリックして選択します。

**手順3:** 画面上部の「**削除**」ボタンをクリックします。

**手順4:** 確認ダイアログで「**削除**」をクリックします。
