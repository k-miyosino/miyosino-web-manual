# 運用者向けマニュアル

ホームページに表示する情報（お知らせ・回覧板・イベントなど）をKintoneで追加・編集・削除する手順を説明します。

## 対象者

- 管理組合事務局のスタッフ
- IT委員会のコンテンツ担当者

## Kintoneとは

**Kintone（キントーン）** はサイボウズ社が提供するデータ管理サービスです。  
ホームページのコンテンツ（お知らせ・回覧板・イベントなど）はすべてKintoneで管理されており、Kintoneに登録した情報が自動的にホームページに反映されます。

## 操作の基本的な流れ

```
Kintoneにログイン → 対象アプリを開く → レコードを追加/編集/削除 → 保存
         ↓
  ホームページに自動反映（リアルタイム）
```

## 各コンテンツの担当アプリ

| ホームページの表示 | Kintoneアプリ名 |
|-----------------|---------------|
| お知らせ | お知らせ |
| 回覧板・配布資料 | 回覧板 |
| イベント情報 | イベント |
| 会議情報 | 会議情報 |
| 各種申請書 | 申請書 |
| グリーンウェルネス | グリーンウェルネス |
